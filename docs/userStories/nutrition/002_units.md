# 002 - units

## Description

As a user, I want to be able to select from predefined units so that I can create recipes efficiently and consistently.

## Acceptance Criteria

- The following units shall be available:
  - g
  - kg
  - Piece(s)
  - tsp
  - tbsp
  - ml
  - cl
  - dl
  - l
  - mm
  - cm
  - To taste
  - Pinch
  - Tip of a knife
  - Handful
  - Slice(s)
